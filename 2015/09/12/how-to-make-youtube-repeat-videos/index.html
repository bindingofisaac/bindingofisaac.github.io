How to make youtube repeat videos

Vivek | 12 Sep 2015

Trying to make youtube videos repeat. something like listeonrepeat but finally i want to make it into an extension for chrome.

by doing view-source on listenonrepeat iframes src had ?enablejsapi=1.
After a quick google search i found this page youtube_api 

function onPlayerStateChange(event) {  
  if (event.data == YT.PlayerState.ENDED) {
    repeat();
  }
}
function repeat() {  
  player.playVideo();
}

So modifying the example in the youtube-api page a little by checking for ending and then playing the video again youtube repeat can be acheieved
