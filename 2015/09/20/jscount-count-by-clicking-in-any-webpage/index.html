jscount :- count by clicking in any webpage

Vivek | 20 Sep 2015

I always had this problem, when I wanted to count something in some website it was difficult to keep track of things with all the colors and scrolling; Not Anymore. By copying this code to chrome console I can keep track of my count. 

var qwerty_wierd_counter = 1;  
document.addEventListener('click', function(e){  
   var div = document.createElement('div');
   var val = document.createTextNode(qwerty_wierd_counter);

   div.appendChild(val);

   div.style.position = 'absolute';
   div.style.left     =  e.pageX + 'px';
   div.style.top      =  e.pageY + 'px';
   div.style.color    =  'red';

   document.body.appendChild(div);
   qwerty_wierd_counter++; 
});
